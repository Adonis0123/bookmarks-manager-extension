# 书签管理器使用指南

## 📦 安装扩展

1. **确保开发服务器正在运行**
   ```bash
   pnpm dev
   ```

2. **在 Chrome 浏览器中加载扩展**
   - 打开 Chrome 浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的 **"开发者模式"**
   - 点击 **"加载已解压的扩展程序"**
   - 选择项目根目录下的 `dist` 文件夹
   - 扩展将会被加载到浏览器中

## 🎯 访问书签管理功能

### 方法 1：通过扩展选项页（推荐）
1. 在 Chrome 扩展管理页面 (`chrome://extensions/`)
2. 找到你刚加载的扩展
3. 点击 **"详细信息"**
4. 点击 **"扩展选项"**
5. 书签管理器界面将在新标签页中打开

### 方法 2：通过扩展图标
1. 点击浏览器工具栏的扩展图标（拼图图标）
2. 找到你的扩展并点击图标
3. 在弹出窗口中，可以看到简单的界面
4. 右键点击扩展图标，选择 **"选项"** 进入书签管理页面

### 方法 3：直接访问 URL
- 在地址栏输入：`chrome-extension://[扩展ID]/options/index.html`
- 扩展 ID 可以在 `chrome://extensions/` 页面找到

## ✨ 功能介绍

### 📊 统计面板
- **总书签数**：显示所有书签的总数
- **文件夹数**：显示文件夹的总数
- **重复书签**：自动检测并显示重复书签数量
- **已选中**：显示当前选中的书签数量

### 🔍 搜索功能
- 在搜索框输入关键词，实时过滤书签
- 支持搜索书签标题和 URL

### 📁 文件夹管理
- 点击文件夹可以展开/收起内容
- 显示每个文件夹中的书签数量
- 支持文件夹的删除操作

### 🎯 书签操作
- **查看书签**：显示书签图标、标题
- **打开书签**：点击外链图标在新标签页打开
- **删除书签**：点击垃圾桶图标删除单个书签
- **批量选择**：使用复选框选择多个书签

### 🔄 拖拽排序
- 按住书签左侧的拖拽图标
- 拖动到目标位置释放
- 支持跨文件夹移动

### 🧹 批量操作
- **全选/取消全选**：一键选择所有书签
- **删除选中**：批量删除选中的书签
- **去重**：自动检测重复书签，一键删除重复项（保留第一个）

### 💾 导出功能
- 点击 **"导出"** 按钮
- 将所有书签导出为 JSON 格式文件
- 文件名包含导出日期

### 🔄 刷新
- 点击刷新按钮重新加载书签数据

## ⚠️ 注意事项

1. **权限要求**：扩展需要 `bookmarks` 权限才能访问书签
2. **数据安全**：
   - 删除操作不可恢复，请谨慎操作
   - 建议在批量删除前先导出备份
3. **去重功能**：
   - 基于 URL 完全匹配检测重复
   - 保留第一个出现的书签，删除后续重复项
4. **拖拽限制**：
   - 某些系统书签（如"书签栏"根文件夹）不能被移动或删除

## 🌙 暗色模式

书签管理器会自动适应系统的暗色模式设置，提供舒适的视觉体验。

## 🐛 问题排查

如果书签管理器无法正常工作：

1. **检查权限**：确保扩展有 `bookmarks` 权限
2. **重新加载扩展**：在扩展管理页面点击刷新按钮
3. **查看控制台**：按 F12 打开开发者工具查看错误信息
4. **重启开发服务器**：停止并重新运行 `pnpm dev`
